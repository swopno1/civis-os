# CivisOS: The Resilience Operating System

> **A "Digital Lifeboat" for Global Survival Scenarios**

In an era of unprecedented global uncertainty, humanity needs a resilient, unkillable network infrastructure. CivisOS is a decentralized "Resilience OS" designed to function when traditional infrastructure fails. We provide the tools to organize, communicate, and survive during severe crises, whether natural, political, or economic.

## 🌟 Vision Statement

CivisOS is more than software—it's a digital lifeboat. When the internet goes dark, when grids collapse, when centralized systems fail or censor, CivisOS ensures that human coordination, knowledge sharing, and survival logistics continue uninterrupted. 

## 🛡️ The 7 Core Facts

1. **Neutrality & Skip Superpower Influence**: CivisOS is politically neutral and explicitly designed to operate independently of any nation-state or corporate control. Code is neutral ground.
2. **Hardware Agnostic**: Runs on whatever you have—from smartphones and old laptops to Raspberry Pis and custom IoT devices.
3. **Poor-First Design**: Optimized for extreme low-bandwidth, low-power, and older hardware. No high-end requirements.
4. **Trusted by ALL**: Fully open-source, verifiable, and transparent to ensure complete trust among all users globally.
5. **Radical Localization**: Built to be easily translated and localized to support marginalized languages and isolated communities.
6. **Offline-First Resilience**: Everything is built to function fully offline. Syncing happens opportunistically when local mesh networks connect.
7. **Modular OS**: A highly extensible plugin system allowing communities to add features tailored to their unique survival needs.

## 🛠️ Core Technology Stack

- **Reticulum Network Stack**: Cryptography-based networking for completely decentralized, un-routable mesh communication.
- **LoRa (Long Range)**: Integration for long-distance, low-bandwidth communication independent of telecom infrastructure.
- **Progressive Web App (PWA)**: Built with Preact & Vite for installable, offline-first applications.
- **RISC-V Compatibility**: Future-proofed for the open hardware revolution.
- **Rust/WASM**: High-performance, memory-safe core logic running anywhere.

## 🎒 The "Everyday Use" Hooks

CivisOS isn't just for doomsday; it's useful right now.

- **Chat Module**: Secure, decentralized messaging that doesn't rely on central servers.
- **Vault Module**: Encrypted local storage for critical documents (IDs, medical records, blueprints).
- **Bulletin Module**: Community notice boards for local coordination, trading, and alerts.
- **Sense Module**: Environmental monitoring and shared sensor data (weather, radiation, air quality).

## 🚀 Getting Started

### For Users

1. **Access the App**: Navigate to our hosted PWA or open a locally hosted node.
2. **Install**: Click "Add to Home Screen" or install the PWA on your device for offline use.
3. **Connect**: Join a local mesh or pair with a LoRa node.
4. **Sync**: You're ready to communicate and access information.

### For Developers

1. **Prerequisites**: Node.js, pnpm, Rust toolchain.
2. **Clone the Repo**: `git clone https://github.com/swopno1/civis-os.git`
3. **Install Dependencies**: `pnpm install`
4. **Start Development**: `pnpm run dev`

See [CONTRIBUTING.md](.github/CONTRIBUTING.md) for full setup and architecture details.

## 🔍 Trust & Verification

Security in CivisOS is life-and-death. 
- **No Backdoors**: Our entire stack, from the networking layer to the UI, is fully open-source.
- **Reproducible Builds**: We provide mechanisms to verify that the binaries we distribute match the source code exactly.
- **AGPL-3.0 Licensed**: To prevent proprietary capture, anyone running CivisOS over a network must share their modifications. See [LICENSE](LICENSE) and [NOTICE](NOTICE).
