# Birds-Eye Plan & Milestones

This document outlines the four-phase development roadmap for CivisOS over its first two years.

## Phase 1: The Foundation (Months 1–4)
* **Goal:** Establish the "Universal Communication Language" and the Web-OS shell.
* **Focus:**
  * Core protocol development using the Reticulum Network Stack.
  * Build the initial web-based "Desktop" UI that is installable and runs in any standard web browser.

## Phase 2: The "Everyday" Utility Suite (Months 5–8)
* **Goal:** User acquisition through daily value.
* **Focus:**
  * Launch the four primary hooks:
    1. **P2P Chat:** Secure, decentralized messaging.
    2. **Personal Vault:** Encrypted local storage for critical documents.
    3. **Local Bulletin Board:** Community notice boards for coordination.
    4. **Offline Maps/Weather:** Crucial environmental data access.

## Phase 3: The Android & Bridge Release (Months 9–12)
* **Goal:** Move from "Web-only" to "System-integrated."
* **Focus:**
  * Launch the native Android APK wrapper (using Capacitor.js or Kotlin).
  * Implement Web-Serial/Bluetooth integration, allowing the browser/app to communicate directly with external LoRa and other radio hardware.

## Phase 4: The Hardware & Pro Launch (Year 2)
* **Goal:** Physical resilience and revenue generation.
* **Focus:**
  * Launch the "Civis-Node" hardware (e-Ink / ESP32 based survival nodes).
  * Open the B2B Module store targeted at NGOs, municipal disaster departments, and local governments.
